Processing a Sale — Step by Step

  1. Go to POS from the sidebar
  2. Search for a product by name/SKU or scan its barcode
  3. If the product has multiple sizes/colors, a variant picker will appear — select the correct one
  4. The item is added to the Cart on the right
  5. Adjust quantities using + / − buttons; apply per-item discount if needed
  6. Optionally attach a Customer using the search field
  7. Apply a cart-level discount amount if applicable
  8. Click Checkout to open the payment panel
  9. Select Payment Method (Cash / Card / Mobile / Bank Transfer)
  10. Enter Amount Paid — change is calculated automatically
  11. Click Complete Sale — a receipt is shown; click Print if needed
Stock Warning A sale will fail if all units of the variant are reserved for rental. Check the Inventory page to ensure sale stock is available (Total Stock − For Rent > 0).

Barcode Scanning

  1. Click inside the barcode scan field (top of product panel)
  2. Scan the product barcode — it auto-searches and adds to cart
  3. If multiple variants match, the variant picker appears
  4. Scan repeatedly to add multiple quantities

Creating a New Rental

  1. Click Rentals in the sidebar → click + New Rental
  2. Search for and select the Customer
  3. Set Rental Start Date and Rental End Date
  4. Choose Event Type (optional — e.g. Wedding, Prom)
  5. Click Add Item — search for a product, select variant and quantity. Repeat for multiple items
  6. Review the Total Rental Cost (auto-calculated: price/day × days)
  7. Enter Advance Payment amount and select payment method
  8. Apply Discount if applicable; add any Notes
  9. Click Create Booking — a confirmation notification is sent to the customer

Processing a Return — Step by Step

  1. Go to Returns in the sidebar
  2. Find the rental and click Process Return
  3. Set the Return Date (defaults to today)
  4. For each item, select its condition using the three tiles:
TileWhen to UseCharge Applied
✓ GoodItem returned in acceptable conditionNo extra charge
⚠ DamagedItem has visible damageAuto-calculated from Settings (flat or % of rental cost)
✕ LostItem not returned / missingSelling price (or custom input for rental-only items)
  1. Review the Charges Summary at the bottom: late fine + damage/loss charges
  2. Select Payment Method for collecting the charges
  3. Click Process Return to confirm
Late Fine Calculation Fine = max(0, days overdue − grace period) × fine per day. The grace period and fine rate are set in Settings → Fine Rules.
Damage Charge Rules Configure in Settings → Fine Rules → Damage Charges. Options: None, Flat Amount (LKR), or Percentage of Item's Rental Cost.

Adding a New Product

  1. Go to Products → click + Add Product
  2. Fill in: Name, SKU, Category, Type (Rental/Sale/Both)
  3. Set Selling Price and Rental Price Per Day
  4. Upload a primary image (drag & drop or click)
  5. In the Variants section, click Add Variant
  6. For each variant: set Size, Color, Material, initial Stock Quantity, and Available for Rent count
  7. Add a Barcode to each variant for POS scanning (optional)
  8. Click Save Product

Printing Barcodes

  1. Open a product's detail page (click any product)
  2. In the Variants table, click the 🖨 Barcode button on any variant row
  3. A preview appears — set the number of copies to print
  4. Click Print — labels are 60mm wide, ready for a label printer

10 · Payroll

Super Admin Manager

Monthly Payroll Workflow

  1. Go to Payroll in the sidebar
  2. Select the month using the month picker at the top
  3. Click Generate Payroll — this creates a record for every active employee (skips existing records)
  4. For each employee, click Edit to open the payroll drawer:
    • Review Base Salary (from HR profile)
    • Add Allowances (e.g. Transport, Performance Bonus) with custom labels and amounts
    • Enter Deductions (e.g. Late Fine)
    • Net Pay = Base + Allowances − Deductions (auto-calculated)
    • Add any Notes
  5. Click Save Changes
  6. When ready to pay, click Pay on each record OR click Pay All to mark all processed records as paid at once

12 · Notifications

Super Admin Manager Cashier

All automated notifications (booking confirmations, pickup reminders, return reminders, late warnings) are logged here. You can also send manual messages.

Sending a Manual Notification

  1. Go to Notifications → click Send Notification
  2. Search and select the Customer
  3. Choose the Channel (SMS, WhatsApp, Email)
  4. Select the Notification Type (e.g. Return Reminder)
  5. Write or edit the Message
  6. Click Send
SMS/WhatsApp Setup Configure FitSMS API Token and Sender ID in Settings → Notifications to enable sending.
